# Recall a Device

Recalling a device un-assigns it from its physical Site. You should perform this action if you are physically moving the device to a new location or removing it from service.

## Steps to Recall

1. In the left sidebar, click **Devices**.
2. Select the specific device you need to recall from the list.
3. On the Device Details page, locate the **Recall Device** button in the top right corner (next to the Add Maintenance Log option).
4. Click **Recall Device**.
5. Confirm the action when prompted.

The device's deployment status will update, and it will no longer be associated with the previous Site coordinates.
