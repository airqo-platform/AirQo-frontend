# Recall a Device

Recalling a device un-assigns it from its physical Site and changes its deployment status to **Recalled**. Perform this action if you're physically moving the device to a new location or taking it out of service.

## Prerequisites

* A device with a deployment status of **Deployed** — the Recall Device option only appears for devices currently assigned to a Site.

## Steps to Recall

1. In the left sidebar, click **My Devices** (Personal Workspace) or **Devices** under Organization Assets (Organization Workspace).
2. Select the device you need to recall from the list.
3. On the Device Details page, click **Recall Device**.
4. Select a **Recall Type**: **Errors** or **Disconnected**.
5. Click **Recall Device** to confirm.

Once recalled, the device's deployment status changes to **Recalled**, and it's no longer associated with its previous Site.
